title: Roadmap
description: What's planned for Browy.
---

Browy is open source — feel free to vote with reactions on
[GitHub issues](https://github.com/BrowyHQ/browy/issues) or open new
ones.

## Now

- Chrome Web Store + Microsoft Edge Add-ons listings (in review)
- macOS and Linux native host builds

## Next

- DevTools panel REPL (run agent commands from the DevTools panel
  alongside your normal Inspector)
- Vision-based action verification (the agent double-checks its own
  click landed where it expected)
- Better single-page-app navigation handling
- Scripted uninstaller

## Later

- Cross-tab workflows (an agent that orchestrates work across multiple
  tabs explicitly)
- Recipe library (save and replay common automations)
- Custom-tool API (write a tool in JS, register it from the side panel)
