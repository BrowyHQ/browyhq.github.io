Browy 由两部分组成：

1. **浏览器扩展**：Chromium 侧边栏加 DevTools 面板。
2. **本地原生消息主机**：一个运行在你机器上的小型 Node.js 辅助进程，
   负责和 GitHub Copilot 通信。

先从 Chrome Web Store 安装扩展，然后运行一行命令安装本地主机。

<Aside type="tip" title="中国大陆开发者请先看这里">
GitHub Releases 和 Copilot 的访问情况和你所处的网络有关。如果安装脚本
卡住或者出现 TLS 错误，参考[中国开发者指南](/zh-cn/china-setup/)里的
镜像方案。
</Aside>

## 第一步：安装扩展

<LinkCard
  title="把 Browy 添加到 Chrome / Edge / Brave"
  href="https://chromewebstore.google.com/detail/iondecjdokngnlkfpipgolgkfegpmjca"
  description="一键安装。同一个商店页面覆盖 Chrome、Edge 和 Brave。"
/>

把 Browy 的工具栏图标固定到工具栏，方便随时打开侧边栏。

<Aside type="tip" title="✓ 确认">
点击 Browy 工具栏图标。侧边栏应该打开，并显示"主机未连接"提示。
这是预期行为，第二步会修复。
</Aside>

<Aside type="tip" title="想用未打包方式加载？">
下面的本地主机安装脚本会同时把扩展的未打包版本放到
`%USERPROFILE%\Downloads\Browy-Extension\`（Windows）或
`~/Downloads/Browy-Extension/`（macOS / Linux）。打开
`chrome://extensions/`，开启**开发者模式**，点击**加载已解压的扩展程序**，
选择那个目录。本地主机已经同时预批准了 Web Store 版本和未打包版本的扩展 ID，
你可以在两者之间随意切换。
</Aside>

## 第二步：安装本地原生消息主机

<Tabs syncKey="os">
<TabItem label="Windows" icon="seti:windows">

打开 PowerShell 运行：

```powershell
irm https://github.com/BrowyHQ/browy/releases/latest/download/install.ps1 | iex
```

安装脚本会：

<Steps>
1. 下载最新 Browy 发行包（约 130 MB）
2. 把本地主机安装到 `%LOCALAPPDATA%\Browy\app\`
3. 在 Chrome / Edge / Brave 注册 `com.browy.host`（同时预批准 Web Store 和未打包扩展 ID）
4. 把扩展的未打包版本放到 `%USERPROFILE%\Downloads\Browy-Extension\`
</Steps>

<Aside type="tip" title="✓ 确认">
重新打开侧边栏。"主机未连接"提示应该消失。如果还在，运行
`& "$env:LOCALAPPDATA\Browy\app\browy.exe" --version` 确认二进制已经装好。
</Aside>

</TabItem>
<TabItem label="macOS" icon="seti:apple">

打开终端运行：

```bash
curl -fsSL https://github.com/BrowyHQ/browy/releases/latest/download/install.sh | bash
```

安装脚本会：

<Steps>
1. 检测你的架构（Apple Silicon 或 Intel）并下载对应的 tar 包（约 125 MB）
2. 把本地主机安装到 `~/.browy/app/`
3. 在 Chrome / Edge / Brave 注册 `com.browy.host`（同时预批准两种扩展 ID）
4. 把扩展的未打包版本放到 `~/Downloads/Browy-Extension/`
</Steps>

<Aside type="tip" title="✓ 确认">
重新打开侧边栏。"主机未连接"提示应该消失。如果还在，运行
`~/.browy/app/browy --version` 确认二进制已经装好。
</Aside>

</TabItem>
<TabItem label="Linux" icon="linux">

打开终端运行：

```bash
curl -fsSL https://github.com/BrowyHQ/browy/releases/latest/download/install.sh | bash
```

安装脚本会：

<Steps>
1. 检测你的架构（`x86_64` 或 `aarch64`）并下载对应的 tar 包（约 133 MB）
2. 把本地主机安装到 `~/.browy/app/`
3. 在 Chrome / Edge / Brave 注册 `com.browy.host`（写入 `~/.config/<browser>/NativeMessagingHosts/`，同时预批准两种扩展 ID）
4. 把扩展的未打包版本放到 `~/Downloads/Browy-Extension/`
</Steps>

<Aside type="tip" title="✓ 确认">
重新打开侧边栏。"主机未连接"提示应该消失。如果还在，运行
`~/.browy/app/browy --version` 确认二进制已经装好。
</Aside>

</TabItem>
</Tabs>

## 第三步：登录 Copilot

点击 Browy 图标打开侧边栏。第一次使用时，Browy 会提示你登录。

<figure class="browy-frame">
  <Image src={panelSignin} alt="Browy 侧边栏显示 'Sign in to GitHub Copilot' 按钮" />
  <figcaption>点击 "Sign in to GitHub Copilot"，然后按提示完成 device-flow 授权。</figcaption>
</figure>

会有一个终端窗口打开，里面有 GitHub Copilot 的 **device-flow** 链接。
把验证码粘贴到链接里、在浏览器里授权 Browy，终端会自动关闭。

<Aside type="tip" title="✓ 确认">
在任意一个标签页里输入"这个页面的标题是什么"。你应该看到一次
`snapshot` 工具调用，然后是流式回复。
</Aside>

完成了。试试[你的第一次对话](/first-chat/)（英文）。

## 更新

再次运行安装脚本即可。它会检测到已有安装并就地升级。

<Tabs syncKey="os">
<TabItem label="Windows" icon="seti:windows">

```powershell
irm https://github.com/BrowyHQ/browy/releases/latest/download/install.ps1 | iex
```

</TabItem>
<TabItem label="macOS / Linux" icon="seti:apple">


</TabItem>

然后在 `chrome://extensions/` 里**重新加载**扩展（点 Browy 卡片上的循环 ↻ 图标）。

## 卸载

<Tabs syncKey="os">
<TabItem label="Windows" icon="seti:windows">

<Steps>
1. 在 `chrome://extensions/` 移除扩展
2. 删除 `%LOCALAPPDATA%\Browy\` 和 `%USERPROFILE%\.browy\`
3. 删除原生消息主机注册：
   `%LOCALAPPDATA%\Google\Chrome\User Data\NativeMessagingHosts\com.browy.host.json`
   （Edge 和 Brave 下也有同名文件）
</Steps>

</TabItem>
<TabItem label="macOS" icon="seti:apple">

<Steps>
1. 在 `chrome://extensions/` 移除扩展
2. 运行内置卸载脚本，然后删除安装目录：
   ```bash
   ~/.browy/app/uninstall.sh && rm -rf ~/.browy ~/Downloads/Browy-Extension
   ```
3. 卸载脚本会清理
   `~/Library/Application Support/Google/Chrome/NativeMessagingHosts/`
   下的清单文件（Edge 和 Brave 也类似）。
</Steps>

</TabItem>
<TabItem label="Linux" icon="linux">

<Steps>
1. 在 `chrome://extensions/` 移除扩展
2. 运行内置卸载脚本，然后删除安装目录：
   ```bash
   ~/.browy/app/uninstall.sh && rm -rf ~/.browy ~/Downloads/Browy-Extension
   ```
3. 卸载脚本会清理 `~/.config/google-chrome/NativeMessagingHosts/`
   下的清单文件（Edge 和 Brave 也类似）。
</Steps>

</TabItem>
</Tabs>
