## 最新

- [**Google 悄悄对开放 web 宣战了。**](/zh-cn/blog/google-vs-the-web/)，2026 年 5 月。
  在 Google I/O 2026 上，AI Overviews 成为搜索的默认体验。
  支撑了 web 二十年的那笔交易被悄悄取消了。Web 现在是原材料。
  中间人在试图把它吞掉。

- [**GitHub 被一个 VSCode 扩展打穿了。3,800 个仓库。**](/zh-cn/blog/vscode-extension-breach/)，2026 年 5 月。
  一个被木马化的 Nx Console 扩展从官方 VS Code 商店出来，
  泄露了 3,800 个内部仓库。更深的问题不在这一个扩展，
  而在于编辑器没有沙箱模型。

- [**每个 $20 的 AI 订阅，实验室要花大约 $100 来服务。账单要来了。**](/zh-cn/blog/ai-subscription-bomb/)，2026 年 5 月。
  AI 实验室在做行业级别的亏本买卖。Anthropic 用户每付 $1 订阅费就
  消耗 $8 的算力。微软在每个 GitHub Copilot 座位上亏 $20+。智能体
  打破了这套数学，6 月 1 日 Copilot 切到按用量计费。

- [**Mullvad 给了你 8 万亿种出口 IP 组合。9 台服务器就把你找出来了。**](/zh-cn/blog/mullvad-vpn-fingerprint/)，2026 年 5 月。
  一位研究者用 3,650 个 WireGuard 密钥反复连上一家声称有数百万种
  出口 IP 组合的 VPN，结果只看到 284 种。原因是用同一个种子浮点数
  在每台服务器的池子里挑索引。

- [**AI slop 杀死了开源漏洞赏金**](/zh-cn/blog/ai-slop-killed-the-bug-bounty/)，2026 年 5 月。
  一家真实的数据库项目刚刚关闭了一个 1,000 美元一个 bug 的赏金
  计划，原因是 AI 智能体往收件箱里塞满了虚构的漏洞报告。成本不在
  Token 上，在维护者的注意力上。
